Please create the following icon files in this directory:

1. app_icon.ico (for Windows)
- Size: 256x256 pixels
- Format: ICO
- Suggested design: A simple microphone icon in purple (hex: #6B4EFF) on a white background

2. app_icon.png (for other platforms)
- Size: 512x512 pixels
- Format: PNG
- Same design as the ICO file

You can use any icon creation tool or download a suitable microphone icon and convert it to the required formats.

For testing purposes, you can use any microphone icon temporarily. The app will work without these icons, but the system tray functionality will be limited.
